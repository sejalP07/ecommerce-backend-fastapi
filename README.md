# E-Commerce Backend API

A RESTful E-Commerce Backend built using FastAPI, SQLAlchemy, JWT Authentication, and PostgreSQL.

## Features

* User Registration
* User Login
* JWT Authentication
* Product CRUD Operations
* Cart Management
* Order Management
* Input Validation
* Swagger API Documentation

## Tech Stack

* FastAPI
* SQLAlchemy
* SQLite / PostgreSQL
* JWT Authentication
* Passlib
* Docker

## Run Project

```bash
uvicorn app.main:app --reload
```
